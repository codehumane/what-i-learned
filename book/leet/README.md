
# 무엇이 1등 팀을 만드는가?

> Leading Effective Engineering Teams

## CH 01. 무엇이 효과적인 팀을 만드는가?

### 1.2 동기부여가 성과를 이끌어낸다

- 내적 동기부여 중요성 이야기
- 여기에는 자율성, 전문성, 목적의식 필요 (다니엘 H. 핑크, 드라이브 인용)
- 데이비드라는 엔지니어를, 수익 보다는 사회적 영향력에 목표를 둔 프로젝트로 이동시켜, 식었던 열정을 되살리기 (목적의식 사례)
- 팀원에게 개별 모듈 개발을 주도하게 해서 자율성과 주인의식 높이도록 (자율성 사례)
- 팀원에게 최신 도구를 접할 수 있게 해 업무에 필요한 기술을 능숙하게 하도록 (전문성 사례)
- 팀원의 기여가 조직 목표에 어떻게 직접적으로 영향을 미치는지 명확히 설명 (목적의식 사례)

### 1.3 효과적인 팀 만들기

#### 1.3.1 적합한 인재 모으기

**적절하게 우선순위를 정하고 독립적으로 업무를 수행**

- 조직과 프로젝트의 전략적 우선순위를 명확하게 공유
- 이를 통해 팀원들이 업무 우선순위를 설정하는 데 도움을 줄 수 있음
- 정기적 체크인 미팅과 목표 설정 회의를 통해 꾸준히 도움을 주고,
- 엔지니어들이 각자 맡은 업무를 효과적으로 처리할 수 있도록 필요한 자원과 지침을 언제나 활용할 수 있도록 준비
